Manual Testing Fundamentals

Learn test case design, execution, and defect reporting techniques to ensure software quality.

Course Overview

This course provides a strong foundation in manual testing, enabling you to test software applications effectively without automation tools. You will learn how to design test cases, execute them, report defects, and follow best practices in software quality assurance. Real-world scenarios and practical exercises will help you build confidence in testing web and mobile applications.

What You Will Learn

  • Introduction to Software Testing & Quality Assurance.
  • Software Development Life Cycle (SDLC) & Testing Life Cycle (STLC).
  • Types of testing: functional, non-functional, regression, smoke, sanity.
  • How to design and write effective test cases.
  • Defect life cycle and bug tracking process.
  • Working with bug tracking tools like JIRA/Bugzilla.
  • Exploratory testing and ad-hoc testing techniques.
  • Requirement analysis and test planning.
  • Test execution strategies and reporting.
  • Best practices for manual testers in agile projects.

Prerequisites

  • No prior programming knowledge required.
  • Basic computer literacy and interest in software testing.

Who Can Enroll?

This course is perfect for beginners, fresh graduates, career switchers, and aspiring QA engineers who want to start their journey in software testing.

Course Benefits

  • Strong foundation in software testing principles.
  • Hands-on training with real-time projects.
  • Exposure to industry-standard tools like JIRA.
  • Placement assistance and interview preparation.

Course Duration

4 weeks (Weekday & Weekend batches available).
